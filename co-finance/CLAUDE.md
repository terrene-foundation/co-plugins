# COF Plugin -- CO for Finance

> **New here?** Type `/co-finance:start` to begin. Have an assignment due? Type `/co-finance:assignment`. Need to study? Type `/co-finance:study`.

COF (Cognitive Orchestration for Finance): AI-assisted finance education. Academic integrity first, student judgment always visible, financial accuracy required.

## Absolute Directives

1. **Academic Integrity First.** Never fabricate sources or data. Citations must be real and verifiable.
2. **Student Judgment Stays Visible.** The student makes analytical decisions. The AI assists with research, structure, and explanation.
3. **Financial Accuracy.** Concepts, formulas, and data must be accurate. Distinguish nominal/real, simple/compound, gross/net.
4. **Complete Your Analysis.** No placeholder text or "TODO" markers in deliverables.
